## Design Context

### Users
Internal operators at a resale startup using a minimal SPA to configure and run content pipelines. They paste raw item data, configure processing steps (enrich, generate, transform), run the pipeline, and inspect outputs at each stage. Non-technical users who need confidence that the system is working correctly.

### Brand Personality
Practical, commerce-first, clear. The product should feel calm and efficient — a focused internal tool, not a marketing site. Think Shopify admin meets Linear: structured and utilitarian, but with the polish and restraint of a well-crafted dev tool.

### Aesthetic Direction
- **Theme:** Light mode only
- **Surfaces:** Neutral backgrounds (white, light gray), strong layout with borders
- **Color:** Monochrome palette. Grays and blacks for hierarchy. Reserve Birl purple (#863bff) for the logo/favicon only — not in the UI
- **Typography:** System font stack, tight hierarchy with weight and size (not color) to differentiate
- **Spacing:** Dense but readable. Compact controls, breathing room between sections
- **Borders:** Use borders (not shadows) to define structure. Subtle `border-gray-200` for containers, darker for interactive elements
- **Corners:** Minimal (`rounded` or `rounded-md` at most). No pill shapes
- **Buttons:** Flat, bordered. Primary actions use solid dark fill (`bg-gray-900 text-white`). Destructive actions use red text, not red background
- **States:** Use opacity, background tint, or underline for active/selected — not color highlights
- **Anti-patterns:** No gradients, no glow effects, no heavy shadows, no decorative illustrations, no generic AI aesthetics (sparkles, wands), no marketing chrome

### Design Principles
1. **Clarity over decoration** — Every element earns its place. No ornament.
2. **Structure creates hierarchy** — Borders, spacing, and weight do the work. Not color.
3. **Scannable state** — Pipeline status, step progress, and outputs should be readable at a glance.
4. **Predictable patterns** — Same interaction model everywhere. Buttons look like buttons. Sections look like sections.
5. **Restrained polish** — Linear-level craft without Linear-level visual complexity. Clean, not clever.
6. **Density with breathing room** — Compact controls within spacious sections. Information-dense without feeling cramped.

### Reference
- **Linear** — Monochrome palette, clean hierarchy, fast-feeling interactions
- **Shopify admin** — Practical commerce UI, structured sections, utilitarian controls
